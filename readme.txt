== Description ==
Video Embed Box plugin is used for video marketing with documentation,details and useful links. 

its only use to embed Videos from
==>Youtube, Vimeo and VideoSuit 
videos with pdf link,audio links and also specify the html content for that video.

Backend options allow the choose favorite colors and icons for the video box.

Look at screenshots or below Demo link for better understanding.

Visit Demo Link [http://topfreelancers.esy.es/classic-video-embeds-box/](http://topfreelancers.esy.es/classic-video-embeds-box/)

== Installation ==
1. Ensure you have compatible version of Wordpress installed
2. Unzip and upload contents of the plugin to your /wp-content/plugins/ directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Left panel have Video Embed Menu to Add new embed and options to change icons and colors for box
5. Add shortcode to page or post to see the result
